# Iterate
2d平台跳跃游戏
开发时间：2026.5-2026.6
简介：本作是一款 PC 端横版轮回探索游戏，玩家操控 AI 数据体在循环关卡中不断轮回迭代，见证 AI 逐步产生自我意识、挣脱操控，最终打破次元壁垒、终结无尽轮回。
说明：此项目基本逻辑基于 CorgiEngine 开发，核心围绕反抗值系统、AI 行为演化、场景动态迭代三大模块进行拓展开发，同时实现周目流程、破壁特效、镜头运镜等配套功能。
其中：
AI 行为与操控逻辑基于引擎原有角色能力模块改造，引入操控权重机制。反抗值随轮回永久累积，数值变化实时调整玩家输入与 AI 自主行为占比，依次实现角色迟疑、逆反行动、自主寻路、屏蔽操控等不同阶段表现。
场景迭代通过场景管理、资源加载类脚本实现，区分玩法结构与美术视觉双维度更新：逐周目修复地图平台、删减陷阱，同步迭代画面画质、色彩、粒子流光与故障特效；所有场景对象预留状态锁定机制，保障轮回切换、视觉特效叠加时运行稳定。
全流程周目体系依托状态机搭建，划分六个递进周目，内置剧情交互关卡与世界观解锁逻辑；高潮阶段结合屏幕碎裂、UI 乱码等破壁特效，完成次元打破的核心表现。
此外，项目还开发了轮回判定逻辑、数字流光特效、动态场景障碍、Cinemachine 虚拟相机运镜、背景视差景深等功能，完整实现玩法、视觉与叙事的整体闭环。
